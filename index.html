Projects

Project-1

DigiVruksh - Indian e-Family Tree
DigiVruksh is an Indian e-Family tree management site designed to store and maintain detailed family lineage information, including Kula, Gotra, Sutra, Kuladevatha, and more. The site features three main modules:
Enter: Allows users to input and save detailed family tree information.
Edit: Provides functionality for users to update and modify existing family records.
Fetch: Enables users to retrieve and view stored family lineage information easily.
This project aims to preserve and digitize traditional family histories in an accessible and user-friendly manner.
